Welcome to Legal-Mails

Legal-Mails is a neutral Electronic Filing Service Provider (EFSP) backend that provides a clean, modern API for e-filing integrations with court systems including Tyler Technologies EFM.

3-Layer Architecture

1. Client Layer (e.g., CallSlayer)

External applications submit cases via our JSON API and poll for status updates.

2. Legal-Mails Core API

Multi-tenant platform with authentication, business logic, PostgreSQL persistence, and document storage.

3. Connector Layer

Integrations with Tyler EFM and other court systems. Generates status events and manages external IDs.

API Endpoints

GET /health

Check API and database health status

POST /api/v1/submissions

Create a new e-filing submission

GET /api/v1/submissions/:id/status

Get submission status with timeline and documents

Authentication

All API endpoints (except /health) require authentication via API key:

X-API-Key: your_api_key_here
